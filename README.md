# Block's Resources - Trazabilidad de Recursos Ambientales en Stellar

Block's Resources es un proyecto blockchain diseñado para la trazabilidad de recursos ambientales en la red Stellar utilizando Soroban, la plataforma de contratos inteligentes de Stellar.

## Características

- **Contrato Inteligente Multisig**: Requiere al menos dos firmas para aprobar transacciones.
- **Trazabilidad de Recursos**: Registro y seguimiento de recursos ambientales.
- **Historial Completo**: Mantenimiento de un historial completo de transferencias y propietarios.
- **Certificación**: Almacenamiento de certificaciones para cada recurso.

## Arquitectura

El proyecto consta de:

1. **Contrato Inteligente**: Implementado en Rust para Soroban.
2. **Scripts de Despliegue**: Para facilitar el despliegue en la red de prueba de Stellar.
3. **Documentación**: Guías detalladas para configurar y ejecutar el proyecto.

## Requisitos

- Rust (última versión estable)
- Stellar CLI
- Soroban CLI
- Cuenta en Stellar Testnet
